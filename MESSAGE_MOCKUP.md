## Technical Details

### HTML Structure
```html
<div class="rounded-md p-4 bg-red-50 dark:bg-red-900/20 border border-red-200 dark:border-red-800"
     x-data="{ show: true }"
     x-show="show"
     x-transition>
    <div class="flex items-start">
        <div class="flex-shrink-0">
            <svg class="h-5 w-5 text-red-600 dark:text-red-400">...</svg>
        </div>
        <div class="ml-3 flex-1">
            <p class="text-sm text-red-800 dark:text-red-300">
                Access Denied: You do not have permission to view Device nodes.
            </p>
        </div>
        <div class="ml-3 flex-shrink-0">
            <button @click="show = false">
                <svg>...</svg> <!-- X icon -->
            </button>
        </div>
    </div>
</div>
```

### CSS Classes Used
- **Background:** `bg-red-50` (light) / `bg-red-900/20` (dark)
- **Border:** `border-red-200` (light) / `border-red-800` (dark)
- **Text:** `text-red-800` (light) / `text-red-300` (dark)
- **Icon:** `text-red-600` (light) / `text-red-400` (dark)

### Alpine.js Directives
- `x-data="{ show: true }"` - Initialize show state
- `x-show="show"` - Control visibility
- `x-transition` - Smooth animations
- `@click="show = false"` - Dismiss on X click
